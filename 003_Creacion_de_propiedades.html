UNIT 003 — Creating Properties (Creación de propiedades, tipos, listas)

What you must know

  • Define multiple properties with clear initial values.
  • Properties can be lists (e.g., phones). (listas, teléfonos)
  • Connect properties to the CRUD menu (insert/read/update/delete).

Example — Product manager with list property

class Product:  # (Producto)
    def __init__(self):
        self.name = ""
        self.price = 0.0
        self.tags = []  # (propiedad lista)

def insert_product(products):
    p = Product()
    p.name = input("Name: ")
    p.price = float(input("Price: "))
    raw = input("Tags (comma separated): ")
    p.tags = [t.strip() for t in raw.split(",") if t.strip()]
    products.append(p)
    print("Inserted.")

def list_products(products):
    for i, p in enumerate(products, start=1):
        print(f"{i} {p.name} €{p.price} tags={p.tags}")

Tip — Arrays/lists as properties

  • Initialize with [] in __init__.
  • When printing, join lists: ", ".join(p.tags).
