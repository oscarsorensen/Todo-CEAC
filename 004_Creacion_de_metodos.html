UNIT 004 — Creating Methods (Creación de métodos, getters, setters, visibilidad)

What you must know

  • Define instance methods that operate on self.
  • Use getters/setters to access “private” data (Python convention uses leading underscore). (getters, setters, privado)
  • Basic validation inside setters (e.g., non-empty name, non-negative price).

Example — Bank account with getters/setters

class BankAccount:  # (Cuenta bancaria)
    def __init__(self):
        self._balance = 0.0  # (privado por convención)

    def deposit(self, amount):          # (método)
        if amount <= 0: return
        self._balance += amount

    def withdraw(self, amount):         # (método)
        if 0 < amount <= self._balance:
            self._balance -= amount

    def get_balance(self):              # (getter)
        return self._balance

    def set_balance(self, value):       # (setter + validación)
        if value >= 0:
            self._balance = value

acc = BankAccount()
acc.deposit(100); acc.withdraw(40)
print(acc.get_balance())  # 60.0

Alternative (Pythonic): @property

Only if allowed by your teacher. Jocarsa often prefers explicit getters/setters.

class Item:
    def __init__(self):
        self._price = 0.0

    @property
    def price(self):
        return self._price

    @price.setter
    def price(self, value):
        if value >= 0: self._price = value
