Los lenguajes de marcado, como HTML y CSS, permiten estructurar, organizar y presentar la información de manera eficiente en la web. Gracias a ellos, es posible definir el formato y el estilo de los documentos, lo que garantiza que el contenido se pueda compartir, reutilizar y mostrar correctamente en diferentes plataformas. En este ejercicio, aplico estos principios al diseño de un CV digital, donde utilizo HTML para estructurar la información y CSS para controlar su aspecto visual, mejorando la claridad y la legibilidad del contenido. Descripción del puesto

Esta tarea muestra cómo se puede utilizar CSS para controlar el diseño visual de un documento HTML, mejorando la estructura y la claridad. Al dividir el CV en secciones y aplicar estilos a cada componente, conseguimos una presentación limpia y profesional.
