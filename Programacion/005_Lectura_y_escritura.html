UNIT 005 — Reading and Writing Information (Lectura y escritura de información, ficheros, pickle, binarios)

This unit explains how to read and write files, both text and binary, and how to save Python objects permanently.

Text files (archivos de texto)

# Write text
with open("notes.txt", "w", encoding="utf-8") as f:
    f.write("First line\nSecond line")

# Read text
with open("notes.txt", "r", encoding="utf-8") as f:
    content = f.read()
print(content)

Appending data

with open("notes.txt", "a", encoding="utf-8") as f:
    f.write("\nAppended line")

Binary files (pickle — binarios)

import pickle

data = ["Oscar", "CEAC", 2025]
with open("data.bin", "wb") as f:
    pickle.dump(data, f)

# Read binary
with open("data.bin", "rb") as f:
    result = pickle.load(f)
print(result)

Exam focus

  • Use open(), write(), read() correctly
  • Know difference between text (r/w/a) and binary (rb/wb)
  • Show saving/loading objects with pickle
