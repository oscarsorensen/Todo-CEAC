En esta tarea, el objetivo del programa es crear una aplicación para un gimnasio online que gestione sesiones de ejercicio. El programa utiliza objetos JavaScript para mostrar los detalles de una sesión de ejercicio, incluyendo la fecha, la hora de inicio, la duración y el tipo de ejercicio. Este ejercicio es relevante para el tema de las propiedades de los objetos en JavaScript, ya que muestra cómo definir, acceder y mostrar propiedades dentro de un objeto. Este programa demuestra cómo utilizar objetos en JavaScript para gestionar información relevante en una aplicación de gimnasio online. Esto podría implementarse fácilmente en un escenario real, para mostrar cuándo, durante cuánto tiempo y qué tipo de formación realizan las personas.
