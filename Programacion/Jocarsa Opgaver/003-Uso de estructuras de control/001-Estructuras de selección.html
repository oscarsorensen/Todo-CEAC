En este ejercicio el objetivo es utilizar correctamente las estructuras de control para clasificar las diferentes etapas de edad. Voy a utilizar var, if, else y document.write. ```` var- Se usa para declarar una variable cuyo valor puede cambiar. Ejemplo: var edad = 25; if- Evalúa una condición. Si se cumple, ejecuta el código dentro de las llaves. Ejemplo: if (edad < 30) { ... } else - Se usa con if para ejecutar otro bloque cuando la condición no se cumple. Ejemplo: else { ... } document.write() - Muestra texto directamente en la página web. Ejemplo: document.write("Eres un joven."); ``` ````` Como puedes ver en el ejercicio, puedes utilizar los conceptos que hemos aprendido, como var, if y else para escribir programas que pueden afectar a nuestra vida diaria.
